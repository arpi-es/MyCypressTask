# My_Cypress_Task
This app fetches data from Api, Saves cache & shows the list of photos Grouped by Albums!
 
## Technologies
- MVVM
- RecyclerView
- Livedata
- Room
- Retrofit
- Hilt
- Databinding
- Navigation

## Features
- Fetches data from api & caches in local database
- Infinity Scroll
- Nested RecyclerView
